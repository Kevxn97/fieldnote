# Chrome Web Store

## Who Or What It Is
Chrome Web Store is an official installation location referenced for Obsidian Web Clipper on Chrome and other Chromium-based browsers.

## Description
In the available material, Chrome Web Store appears as a browser extension distribution channel rather than as a standalone product with its own features described in detail. It is one of the official places users are directed to install Obsidian Web Clipper.

The Chrome listing is explicitly described as covering Chrome, Brave, Arc, Orion, and other Chromium-based browsers. That makes it the main install path for the Chromium browser family in this context.

Its relevance is practical: users who want Obsidian Web Clipper on a Chromium-based browser are pointed to the Chrome Web Store listing to obtain it. No further details are provided here about store behavior, policies, permissions, or update mechanics.

## Key Relationships
- **Obsidian Web Clipper** — listed as an official installation source for the extension.
- **Chrome / Chromium-based browsers** — the install path is said to cover Chrome, Brave, Arc, Orion, and other Chromium browsers.
- **Obsidian Help** — the introduction page directs users to the Chrome Web Store for installation.
- **Browser extension distribution** — functions as a distribution point for the Chromium version of the tool.
- **Firefox Add-Ons / Safari Extensions / Edge Add-Ons** — appears alongside other official extension storefronts for other browser ecosystems.
