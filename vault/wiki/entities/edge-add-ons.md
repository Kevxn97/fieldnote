# Edge Add-Ons

## Who Or What It Is
Edge Add-Ons is an official installation location listed for Obsidian Web Clipper, indicating it is a browser extension distribution channel associated with Microsoft Edge.

## Description
Edge Add-Ons appears as one of several browser-specific storefronts or extension catalogs where Obsidian Web Clipper can be installed. It is listed alongside Chrome Web Store, Firefox Add-Ons, and Safari Extensions.

Its significance here is distribution rather than product behavior. For Obsidian Web Clipper, Edge Add-Ons serves as the Edge-specific path for obtaining the extension.
