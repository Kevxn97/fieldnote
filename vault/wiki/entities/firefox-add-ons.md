# Firefox Add-Ons

## Who Or What It Is
Firefox Add-Ons is an official installation location linked from Obsidian Help for obtaining Obsidian Web Clipper on Firefox. In the cited material, it is associated with support for both Firefox and Firefox Mobile.

## Description
Firefox Add-Ons appears as one of several browser-specific distribution points for Obsidian Web Clipper. Obsidian Help lists it alongside Chrome Web Store, Safari Extensions, and Edge Add-Ons as a place users can install the extension.

Within that usage, Firefox Add-Ons matters as the Firefox-family path for accessing Obsidian Web Clipper. The linked Firefox listing is explicitly described as supporting Firefox and Firefox Mobile, which distinguishes it from the Chromium, Safari, and Edge distribution channels.

The material does not describe Firefox Add-Ons in broader platform terms. Its evidenced role is specifically as the Firefox install route for a free browser extension that highlights pages and saves web content to an Obsidian vault.

## Key Relationships
- Linked from **Obsidian Help** as an official install location for **Obsidian Web Clipper**.
- Associated with support for **Firefox**.
- Associated with support for **Firefox Mobile**.
- Listed alongside **Chrome Web Store**, **Safari Extensions**, and **Edge Add-Ons** as part of Web Clipper’s browser distribution.
- Connected indirectly to **Obsidian vaults** through Obsidian Web Clipper, which saves clipped content locally.
