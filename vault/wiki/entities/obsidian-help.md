# Obsidian Help

## Who Or What It Is
Obsidian Help is the documentation surface referenced by the “Introduction to Obsidian Web Clipper” page. In this material, it functions as an official help resource that introduces Obsidian Web Clipper, links to installation options across major browsers, and organizes related feature and troubleshooting documentation.

## Description
Obsidian Help appears here as a structured product documentation hub rather than a standalone application or service. The cited page introduces Obsidian Web Clipper as a free browser extension for highlighting pages and saving web content directly to an Obsidian vault.

Its role is strongly navigational. The Web Clipper introduction page links users to official installation locations for Chrome/Chromium browsers, Firefox, Safari, and Edge, and also routes them to topic-specific documentation such as clipping, highlighting, reader mode, interpreter features, troubleshooting, templates, variables, filters, and logic.

The help content also carries product-positioning information. In this case, it states that clipped content is saved locally to the user’s Obsidian vault, that no user data or usage metrics are collected, and that the codebase is open source and auditable. That makes Obsidian Help a source not just of usage instructions, but also of privacy and transparency claims about Obsidian Web Clipper.

## Key Relationships
- **Obsidian Web Clipper** — Obsidian Help introduces and documents this browser extension.
- **Obsidian vault** — Obsidian Help states that Web Clipper saves content locally to the user’s vault.
- **Chrome Web Store / Chromium browsers** — Obsidian Help provides an official installation route covering Chrome, Brave, Arc, Orion, and other Chromium-based browsers.
- **Firefox Add-Ons** — Obsidian Help links Firefox installation and notes support for Firefox and Firefox Mobile.
- **Safari Extensions** — Obsidian Help links Safari installation and notes support for macOS, iOS, and iPadOS.
- **Edge Add-Ons** — Obsidian Help provides Edge installation access for Web Clipper.
- **Feature documentation** — Obsidian Help links subtopics for Clip web pages, Highlighter, Reader, Interpreter, Troubleshooting, Templates, Variables, Filters, and Logic.
- **obsidianmd/obsidian-clipper** — Obsidian Help connects Web Clipper’s privacy/transparency claims to an open-source, auditable codebase.
