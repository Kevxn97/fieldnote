# Obsidian Web Clipper

## Who Or What It Is
Obsidian Web Clipper is a free browser extension for highlighting pages and saving web content directly to an Obsidian vault.

## Description
The extension is presented as a web-clipping tool integrated with Obsidian’s local vault model. Its core purpose is to capture content from the web and store it in the user’s vault rather than in a cloud service.

It is distributed across multiple browser ecosystems. Official installation locations are listed for Chrome/Chromium browsers, Firefox, Safari, and Edge, with the Chrome listing covering Chrome, Brave, Arc, Orion, and other Chromium-based browsers. Firefox support explicitly includes Firefox Mobile, and Safari support explicitly includes macOS, iOS, and iPadOS.

The documented feature surface extends beyond simple page saving. Linked help topics indicate support for clipping web pages, highlighting, reader mode, an Interpreter feature, troubleshooting, templates, variables, filters, and template logic.

A major distinguishing characteristic is its privacy and transparency stance. Content is said to be saved locally to the user’s Obsidian vault, user data is not collected, usage metrics are not gathered, and the codebase is described as open source and auditable.
