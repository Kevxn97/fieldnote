# obsidianmd/obsidian-clipper

## Who Or What It Is
`obsidianmd/obsidian-clipper` appears to be the open-source, auditable code repository associated with Obsidian Web Clipper, a free browser extension for highlighting pages and saving web content directly to an Obsidian vault.

## Description
The repository is referenced as the codebase behind Obsidian Web Clipper’s privacy and transparency claims. Obsidian Help states that the clipper saves content locally to the user’s vault, does not collect user data, does not gather usage metrics, and is open source and auditable.

Its importance is tied to trust and inspection rather than to a feature list in the available material. The linked repository supports the claim that the extension is transparent and can be examined, instead of operating as a closed or cloud-dependent service.

The surrounding documentation describes the product supported by this repository as a browser extension available across Chrome/Chromium browsers, Firefox, Firefox Mobile, Safari, macOS, iOS, iPadOS, and Edge. Those documents also connect the codebase to features such as clipping web pages, highlighting, reader mode, interpreter features, troubleshooting, templates, variables, filters, and logic.

The available evidence does not describe the repository’s internal structure, maintainers, license, release process, or implementation details. Its role is clear, but its contents remain only lightly characterized.
