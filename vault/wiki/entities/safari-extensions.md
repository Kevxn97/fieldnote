# Safari Extensions

## Who Or What It Is
Safari Extensions is the Safari distribution channel listed for installing Obsidian Web Clipper. In this context, it refers to the official Safari install location for the browser extension, with support explicitly noted for macOS, iOS, and iPadOS.

## Description
Within the Obsidian Web Clipper ecosystem, Safari Extensions is one of the official browser-specific install endpoints alongside Chrome Web Store, Firefox Add-Ons, and Edge Add-Ons. Its role is straightforward: it is where Safari users obtain the extension.

The Safari listing is notable because it is explicitly associated with Apple’s desktop and mobile platforms. The installation path is described as supporting macOS, iOS, and iPadOS, which distinguishes it from the Firefox and Chromium-oriented listings mentioned elsewhere.

Its importance comes from platform reach. Obsidian Web Clipper is presented as a free browser extension for highlighting pages and saving web content directly to an Obsidian vault, and Safari Extensions is the route that makes those capabilities available to Safari users across Apple devices.

The available material does not describe Safari Extensions as a standalone product beyond that install role. It also does not specify whether the Safari version differs functionally from the Chrome, Firefox, or Edge versions.

## Key Relationships
